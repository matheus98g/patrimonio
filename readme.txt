resumo
- controle de ativos da sala de manutenção/informatica do SENAC

features
- banco de dados
- controle de acesso
- cadastro e alteração de informações de usuarios
- cadastro e alteração de informações de ativos
- listar, editar, ativar/inativar ativos
- pesquisa de ativos
- controle/historico de movimentações de ativos (usuario, operação, onde saiu, onde entrou, motivo, datahora)


Funcionalidades
Banco de Dados: Definir as relações entre as tabelas, como um usuário pode ter várias movimentações de ativos.
Controle de Acesso: Usar o sistema de autenticação do Laravel para restringir o acesso às funcionalidades com base nos perfis de usuários.
Cadastro e Alteração: Formulários para inserir e atualizar informações de usuários e ativos.
Listagem e Pesquisa: Implementar funcionalidades para listar ativos com opções de filtro e pesquisa.
Histórico de Movimentações: Registrar cada movimentação de ativo com detalhes como usuário, operação, localização e data/hora.
